# Chicken Run
Play Chicken Run with DQN.

### Prerequisites
* Keras
* TensorFlow
* NumPy
* Pygame
* scikit-image

### Run
* Train the model:
```
    python3 dqn.py -m Train
```
* Run the program using trained model:
```
    python3 dqn.py -m Run
```
* Play it yourself!
```
    python3 dqn.py -m Play
    Press any key to jump
```
